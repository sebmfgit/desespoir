# README

## créer une branche
```bash
git branch fleur
```

## changer de branche
```bash
git checkout fleur
```

## créer une branche ET changer en même temps
```bash
git checkout -b rose
``` 
## fusionner deux branches
```bash
git checkout master
git merge fleur
```

## lister les branches
```bash
git branch
```
